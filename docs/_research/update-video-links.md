## Overview

The current README links three GitHub Universe 2021 YouTube videos that are now significantly outdated.
GitHub has undergone massive changes since 2021—Copilot, Agent HQ, multi-model choice, agentic
code review, and a complete GitHub Actions re-architecture. The resources below replace the 2021 videos
with current (2024–2026) content organized by GHEC administration topic.

**Key finding:** GitHub Universe 2024 and 2025 session recordings are available on the
[GitHub YouTube channel](https://www.youtube.com/@GitHub) but individual video IDs for keynotes
could not be programmatically verified (YouTube blocks scraping). The official recap page and blog
posts below are verified and link directly to announcements. YouTube videos from the GitHub channel
that *were* verified via the oEmbed API are marked ✅; others are marked `[NEEDS VERIFICATION]`.

## Certification Resources

GitHub offers five official certifications, all administered through Microsoft Learn:

| Certification | Exam | URL | Relevance to GHEC Admin |
|--------------|------|-----|------------------------|
| GitHub Foundations | GH-900 | <https://learn.microsoft.com/en-us/credentials/certifications/github-foundations/> | Foundational — good for all team members |
| GitHub Administration | GH-100 | <https://learn.microsoft.com/en-us/credentials/certifications/github-administration/> | **Primary cert** for GHEC admins — repo mgmt, workflow optimization, enterprise admin |
| GitHub Actions | GH-200 | <https://learn.microsoft.com/en-us/credentials/certifications/github-actions/> | CI/CD automation, workflow management |
| GitHub Copilot | GH-300 | <https://learn.microsoft.com/en-us/credentials/certifications/github-copilot/> | Copilot admin controls, enterprise governance |
| GitHub Advanced Security | GH-500 | <https://learn.microsoft.com/en-us/credentials/certifications/github-advanced-security/> | Secret scanning, code scanning, supply chain security |

All certifications are valid for two years. Practice assessments are available for GH-300 and GH-500.
GitHub Universe events sometimes include on-site certification testing (Universe 2025 had Day 3 testing).

### Note on YouTube Video Discovery

YouTube's client-side rendering blocks programmatic content extraction. Specific video IDs for GitHub
Universe 2024 and 2025 keynote recordings were not discoverable via the RSS feed (which only returns
the 15 most recent uploads) or web scraping. The blog posts and recap page provide equivalent or
superior written content. To find keynote recordings, search "GitHub Universe 2024 keynote" or
"GitHub Universe 2025 keynote" on the [@GitHub YouTube channel](https://www.youtube.com/@GitHub).
